# DataWise Desktop 统一视觉规范
版本：v1.0  
日期：2025-11-14  
适用范围：Tauri、egui、tui 三个 UI 框架

---

## 1. 颜色规范

### 1.1 主色调

| 用途 | Tauri | egui | tui |
|------|-------|------|-----|
| 背景 | `#1e1e1e` | `#252525` | 黑色 (0) |
| 前景 | `#e0e0e0` | `#e0e0e0` | 白色 (15) |
| 强调 | `#007acc` | `#00ff9f` | 绿色 (10) |
| 成功 | `#4ec9b0` | `#4ec9b0` | 绿色 (10) |
| 警告 | `#dcdcaa` | `#dcdcaa` | 黄色 (11) |
| 错误 | `#f48771` | `#f48771` | 红色 (9) |

### 1.2 进度条颜色

- **Tauri**：蓝色 `#007acc`
- **egui**：绿色 `#00ff9f`
- **tui**：绿色 (10)

### 1.3 按钮颜色

| 状态 | 背景 | 文字 |
|------|------|------|
| 正常 | `#007acc` | 白色 |
| 悬停 | `#005a9e` | 白色 |
| 按下 | `#004578` | 白色 |
| 禁用 | `#404040` | `#808080` |

## 2. 字体规范

### 2.1 字体选择

| 用途 | Tauri | egui | tui |
|------|-------|------|-----|
| 编辑器 | `Consolas` / `Monaco` | `UbuntuMono-Regular` | `Monospace` |
| UI 文本 | `Segoe UI` / `SF Pro` | `UbuntuMono-Regular` | `Monospace` |
| 标题 | `Segoe UI` / `SF Pro` | `UbuntuMono-Regular` | `Monospace` |

### 2.2 字体大小

| 元素 | 大小 | 说明 |
|------|------|------|
| 标题 | 16px / 18px | Header 标题 |
| 正文 | 12px / 13px | 编辑器、表格 |
| 小字 | 11px / 12px | 状态栏、提示 |

### 2.3 行高

- 编辑器：1.5
- 表格：1.4
- 正文：1.6

---

## 3. 布局规范

### 3.1 间距

| 元素 | 间距 |
|------|------|
| 组件间距 | 8px |
| 内边距 | 12px |
| 外边距 | 16px |

### 3.2 圆角

| 元素 | 圆角 |
|------|------|
| 按钮 | 4px |
| 输入框 | 4px |
| 卡片 | 8px |

### 3.3 边框

| 元素 | 宽度 | 颜色 |
|------|------|------|
| 输入框 | 1px | `#404040` |
| 卡片 | 1px | `#404040` |
| 焦点 | 2px | `#007acc` / `#00ff9f` |

---

## 4. 进度条规范

### 4.1 尺寸

- **高度**：4px（Tauri/egui）、1 行（tui）
- **宽度**：100%（全宽）

### 4.2 动画

- **Tauri**：线性动画，100ms 更新一次
- **egui**：线性动画，100ms 更新一次
- **tui**：无动画，100ms 更新一次

### 4.3 显示内容

```
[████████░░░░░░░░] 45% | 234 MB / 512 MB | ETA: 2m 30s
```

---

## 5. 错误提示规范

### 5.1 Toast 提示

- **位置**：右下角（Tauri/egui）、底部（tui）
- **持续时间**：3 秒（成功）、5 秒（错误）
- **样式**：圆角卡片，带图标

### 5.2 错误信息

- **颜色**：红色 `#f48771`
- **字体**：等宽字体，便于复制
- **位置**：Log 页，带行号

---

## 6. 表格规范

### 6.1 列宽

- 自适应内容宽度
- 最小宽度：60px
- 最大宽度：400px

### 6.2 行高

- 默认：32px
- 紧凑：24px
- 宽松：40px

### 6.3 交替行颜色

- 奇数行：`#1e1e1e`
- 偶数行：`#252525`

---

## 7. 图标规范

### 7.1 图标大小

| 用途 | 大小 |
|------|------|
| 按钮 | 16px |
| 菜单 | 16px |
| 状态 | 12px |

### 7.2 图标风格

- 线性风格，2px 笔画
- 单色，与文字同色

---

## 8. 响应式设计

### 8.1 最小窗口尺寸

- **Tauri**：1024×768
- **egui**：800×600
- **tui**：80×24

### 8.2 自适应布局

- 窗口宽度 < 1200px：隐藏侧边栏
- 窗口高度 < 600px：隐藏 Chart 页

---

## 9. 深色/浅色主题

### 9.1 当前支持

- 仅深色主题（v1.0）

### 9.2 未来计划

- v0.3 支持浅色主题
- 自动跟随系统主题

---

## 10. 无障碍设计

- 最小字体：11px
- 最小按钮：32×32px
- 对比度：WCAG AA 标准
- 焦点指示：2px 蓝色/绿色边框
